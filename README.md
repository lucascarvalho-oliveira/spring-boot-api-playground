# Spring Boot API Playground

Repositório dedicado aos meus estudos de **Spring Boot** e desenvolvimento de **APIs REST** em Java.

O objetivo deste repositório é reunir pequenos projetos e exemplos práticos para aprender os principais conceitos do ecossistema Spring Boot.

## Tecnologias

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Hibernate
- Maven
- H2 Database
- MySQL
- REST API

## Conceitos estudados

- Estrutura de um projeto Spring Boot
- Controllers
- Services
- Repositories
- Entidades (JPA)
- CRUD
- Injeção de Dependência
- Spring Data JPA
- Banco de dados H2
- MySQL
- Tratamento de exceções
- Validação de dados
- Arquitetura em camadas
- API REST

## Objetivo

Este repositório faz parte da minha jornada de aprendizado em desenvolvimento Back-end com Java e Spring Boot. Cada projeto explora um conjunto específico de funcionalidades e boas práticas para construção de APIs REST.
