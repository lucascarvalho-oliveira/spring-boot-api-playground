# Spring Boot API Playground

Repository dedicated to my **Spring Boot** and **Java REST API** studies.

The purpose of this repository is to gather small projects and practical examples to learn the main concepts of the Spring Boot ecosystem.

## Technologies

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Hibernate
- Maven
- H2 Database
- MySQL
- REST API

## Concepts Studied

- Spring Boot project structure
- Controllers
- Services
- Repositories
- Entities (JPA)
- CRUD
- Dependency Injection
- Spring Data JPA
- H2 Database
- MySQL
- Exception Handling
- Data Validation
- Layered Architecture
- REST API

## Goal

This repository is part of my learning journey in Back-end development with Java and Spring Boot. Each project explores a specific set of features and best practices for building REST APIs.
